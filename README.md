### Quick start

```bash

# clone the repo
$ git clone https://github.com/petersandor/books-app-client.git

# go into the directory
$ cd books-app-client

# install npm dependencies
$ npm install

# start the server
$ npm start
```

go to [http://localhost:9000](http://localhost:9000) in your browser.
